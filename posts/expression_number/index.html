숫자의 표현

문제설명 #

  1. 자연수 n을 연속된 자연수들의 합으로 표현하는 방법을 찾는 문제이다.
    예: 15 → 4가지 방법

    • 1 + 2 + 3 + 4 + 5 = 15
    • 4 + 5 + 6 = 15
    • 7 + 8 = 15
    • 15 = 15

  2. 매개변수로 자연수 n이 주어진다.
    제한사항: n은 10,000 이하의 자연수이다.

  3. n을 연속된 자연수들의 합으로 표현하는 방법의 개수를 반환하는 solution 함수를 작성한다.1

문제 나누기 #

처음에는 $10000$까지의 자연수를 모두 더하여 반복문을 통해 순회를 하면서 $element - n$을 만족하는 수를 std::find(s.begin(), s.end(), element-n)을 통해 찾으려 하였다. 하지만 해당 방식으로 문제를 풀면 효율성 검사에서 실패하게 된다. for문과 std::find()를 동시에 쓴다면 시간 복잡도는 $O(n^2)$가 되기 때문이다. 따라서 다른 방식을 생각하여 문제를 풀어야 했다.

해당 문제는 등차 수열의 합을 구하는 공식을 떠올려 풀어야 효율성 검사를 통과할 수 있었다.

문제 해결하기 #

아래는 시작값이 $a$이고 등차가 $1$인 등차 수열의 합을 구하는 공식을 유도하는 과정이다.

$$ \begin{align*} n &= a + (a+1) + (a+2) + \cdots + (a+k-1) \newline \newline n &= ka + \frac{(k-1)(k)}{2} \newline \newline ka &= n - \frac{(k-1)(k)}{2} \newline \newline a &= \frac{n - \frac{(k-1)(k)}{2}}{k} \end{align*} $$

여기서 a가 자연수라면 초기값이 a이고 k항을 가진 등차가 1인 수열이 존재함을 의미한다. 따라서 a가 자연수인 k를 구하면 해당 숫자는 연속된 자연수들의 합으로 표현할 수 있다.

자연수이기 위한 조건은 $a$가 양수이고 $n-\frac{(k-1)\cdot k}{2}$가 $k$의 배수여야 한다.

전체코드 # 

int solution(int n){
   int answer = 0;
   while(n > (k-1)*k/2){ // a값이 양수일 조건
      if(n-(k-1)*k/2 % k == 0) { // k의 배수인 경우
         answer++; // 연속된 자연수의 합으로 표현될 수 있다.
      }
   } 
   return answer;
}
