클라이언트로부터 서버로 #

CLIENT에서 SERVER로 데이터를 전송하는 방식은 크게 2가지가 존재한다.

  1. 쿼리 파라미터를 통한 데이터 전송
    • GET
    • 주로 정렬 필터
  2. 메시지 바디를 통한 데이터 전송
    • POST, PUT, PATCH
    • 리소스 변경

CLIENT에서 SERVER로 데이터를 전송할 상황은 크게 4가지이다.

  • Static Data 조회
    • 이미지, 텍스트 문서
  • Dynamic Data 조회
    • 검색, 게시판에서 정렬 필터(검색어)
  • HTML Form을 통한 데이터 전송
    • 회원 가입, 상품 주문, 데이터 변경
  • HTTP API를 통한 데이터 전송
    • 회원 가입, 상품 주문, 데이터 변경
    • Server to Server, App Client, Web Client(Ajax)
